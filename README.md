# LAMCODE
My first bigger open-source project. The app is designed to teach people basic stuff about programming &amp; more.

### Note about cloning
File `config.json` is private, use the following schema for config.json.
```json
{
    "id": "",
    "clientSecret": "",
    "callbackURL": ""
}
```
